## 添加卡
#### 用途描述: 添加绑定银行卡
#### URL:  secapi/user/bindCardOnly

#### 输入:
| 字段名          | 字段含义     | 类型     | 最大长度 | 必须   | 说明   |
| :----------- | :------- | :----- | :--- | :--- | :--- |
| cardNo       | 卡号       | string |      | M    |      |
| identity     | 证件号码     | string |      | M    |      |
| identityType | 证件类型     | string |      | M    |      |
| mobile       | 银行卡预留手机号 | string |      | M    |      |
| openID       | 用户标识     | string |      | M    |      |
| realName     | 账户真实姓名   | string |      | M    |      |
#### 输出:
| 字段名     | 字段含义       | 类型     | 最大长度 | 必须   | 说明      |
| :------ | :--------- | :----- | :--- | :--- | :------ |
| bindFlg | 该卡之前是否已经绑定 | string |      |      | bindFlg |
