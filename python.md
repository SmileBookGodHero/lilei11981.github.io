### Python基础

* 数据类型和变量
    * 整数：1，100，-2，0，0xff00（十六进制数），0xa5b4c3d2 (十六进制数)
    * 浮点数：1.23，3.14，-9.01，1.23e5（123000），1.2e-5(0.000012)
    * 字符串：'abc'，"xyz"
    * 布尔值：True、False
    * 空值：NONE
    * 变量：变量名必须是大小写英文、数字和```_```的组合，且不能用数字开头
    * 常量：PI = 3.14159265359

* 字符串和编码
> 对于单个字符的编码，Python提供了ord()函数获取字符的整数表示，chr()函数把编码转换为对应的字符：

```
>>> ord('A')
65
>>> ord('中')
20013
>>> chr(66)
'B'
>>> chr(25991)
'文'
```
